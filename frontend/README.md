# Suuka Cleaning Frontend

This sample frontend is a React + TypeScript landing page for the Suuka cleaning platform.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

## Notes for developers

- Use this structure as the basis for a client portal and enterprise dashboard.
- Add API integration for bookings, cleaner assignments, consultations, products, and approvals.
- The frontend is intentionally small and easy to extend.
